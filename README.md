#### 사용하기

1. 이 저장소 clone 하기 

2. npm install   

3. config 폴더 생성, 아래에 .env파일 생성
```
    MONGODBURI = ( 몽고 db uri )
   
    KAKAOKEY = ( 카카오 앱키 [javascript 키] )
    
    NAVERKEY = ( 네이버 앱키 [)
```
4. npm start
